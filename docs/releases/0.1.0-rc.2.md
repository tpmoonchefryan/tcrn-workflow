# TCRN Workflow 0.1.0-rc.2

`0.1.0-rc.2` is an immutable unpublished local Workflow release candidate.
It repairs the CI bootstrap so pinned Node runs before a pinned pnpm `11.3.0`
acquisition with an explicit online override. Project dependency acquisition
remains frozen and lifecycle-script-free, and P1 verification remains offline.

This candidate is not published, is not a supported AOS release, has
`supportedAosReleases: []`, and does not authorize remote, AOS, connected-mode,
or publication operations. The prior `0.1.0-rc.1` candidate remains immutable.
