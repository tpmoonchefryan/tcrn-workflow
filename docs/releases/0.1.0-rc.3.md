# TCRN Workflow 0.1.0-rc.3

`0.1.0-rc.3` is an immutable unpublished local Workflow release candidate.
It repairs the P3 stale-lease recovery proof so it is valid when a filesystem
reuses a removed directory inode. The proof binds the quarantined pathname and
generation, preserves stale-entry non-carryover, and retains fail-closed
single-writer recovery.

This candidate is not published, is not a supported AOS release, has
`supportedAosReleases: []`, and does not authorize remote, AOS, connected-mode,
or publication operations. The prior `0.1.0-rc.1` and `0.1.0-rc.2` candidates
remain immutable.
