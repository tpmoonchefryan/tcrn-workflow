# Support

The project is pre-release and community supported. `0.1.0-rc.4` is an
unpublished local candidate with no supported AOS release; the prior
`0.1.0-rc.1`, `0.1.0-rc.2`, and `0.1.0-rc.3` candidates remain immutable and are
superseded for current-candidate purposes. Use a discussion for usage questions
and an issue for reproducible defects. Security reports follow `SECURITY.md`.

Development mode is supported only with the pinned toolchain. Release mode is
supported only when the external trust-root verification contract succeeds.
